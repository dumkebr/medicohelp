Aqui vai o código HTML completo, prontinho pra colar no Replit (ou direto no componente React/HTML da home do MédicoHelp).
O áudio pode ser local (audio/alo_doutor.mp3) ou puxado de uma URL.

<section class="text-white text-center py-16 bg-gradient-to-br from-emerald-900 to-emerald-800 rounded-2xl shadow-lg">
  <h3 class="text-sm tracking-widest text-emerald-400 uppercase mb-2">Apresentamos a Dra. Clarice</h3>
  <h1 class="text-4xl font-bold mb-4">Decisão clínica rápida, do jeito tradicional — feita por médicos, para médicos.</h1>
  <p class="max-w-2xl mx-auto text-lg text-gray-300 mb-6">
    A Dra. Clarice apoia sua conduta com respostas objetivas (<strong>Modo Clínico</strong>), explicações baseadas em evidências e ferramentas práticas
    (<strong>MedPrime</strong>: calculadoras e posologia). Clareza, segurança e medicina de verdade — do jeito que sempre funcionou.
  </p>

  <p class="text-emerald-300 font-semibold mb-8">
    💬 Sabia que você pode até “ligar” para a Dra. Clarice?  
    Clique abaixo e ela te atende pessoalmente!
  </p>

  <div class="flex justify-center gap-4">
    <button id="ligarClarice" class="bg-emerald-500 hover:bg-emerald-600 text-white px-6 py-3 rounded-full text-lg font-semibold shadow-md transition-all duration-300 flex items-center gap-2">
      📞 Ligar para a Dra. Clarice
    </button>
    <button class="border border-emerald-500 hover:bg-emerald-700 text-white px-6 py-3 rounded-full text-lg font-semibold transition-all duration-300">
      Como funciona
    </button>
  </div>

  <!-- Áudio da voz da Dra. Clarice -->
  <audio id="audioClarice">
    <source src="audio/alo_doutor.mp3" type="audio/mpeg">
    Seu navegador não suporta áudio.
  </audio>

  <script>
    const botao = document.getElementById('ligarClarice');
    const audio = document.getElementById('audioClarice');

    botao.addEventListener('click', () => {
      botao.textContent = '🔔 Ligando para a Dra. Clarice...';
      setTimeout(() => {
        botao.textContent = '📞 Conversando com Dra. Clarice';
        audio.play();
      }, 1800); // Simula o tempo da chamada antes de atender
    });
  </script>
</section>


👉 Como usar:

Crie uma pastinha audio dentro do seu projeto.

Coloque lá o arquivo alo_doutor.mp3 com a fala:
“Alô, doutor, tudo bem? Estou aqui para te ajudar.”

O botão toca o som e muda o texto simulando a ligação.
